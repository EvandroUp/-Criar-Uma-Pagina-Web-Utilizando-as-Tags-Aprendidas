Tag "h1 à h6"

Usamos as tags "H + Um número de 1 a 6" para títulos, onde o h1 dá uma fonte maior para o título e a 6 é a menor do título. exemplo: Usamos o "H + um número de 1 a 6" dentro de "< >" e "< / >" para que seja aberto um lugar em que colocamos um título e entre os mesmo colocamos o que queremos escrever, na maioria das tags fazemos isso.

Tag "p"

Usamos a tag "p" para escrever um parágrafo.

Tag "mark"

Usamos a tag "mark" para fazer uma marcação no texto.

Tag "i"

O elemento HTML (i) representa uma gama de texto que é separada do texto normal por algum motivo, como texto idiomático, termos técnicos, designações taxonômicas, entre outros. Historicamente, estes têm sido apresentados usando o tipo itálico, que é a fonte original da nomeação deste elemento.

Tag "u"

O elemento HTML (u) representa uma extensão de texto embutido que deve ser renderizado de uma forma que indique que ele tem uma anotação não textual. Isso é renderizado por padrão como um sublinhado sólido simples, mas pode ser alterado usando CSS. Esse elemento costumava ser chamado de elemento "Sublinhado" em versões mais antigas do HTML, e ainda é às vezes mal usado dessa maneira. Para sublinhar o texto, você deve aplicar um estilo que inclua a propriedade de decoração de texto CSS definida como .underline

Tag "strong"

O elemento HTML (strong) indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito.

Tag "ol"

O elemento HTML (ol) representa uma lista ordenada de itens — normalmente renderizada como uma lista numerada.

Tag "ul"

O elemento HTML (ul) representa uma lista não ordenada de itens, normalmente renderizada como uma lista com marcadores.

Tag "li"

O elemento HTML (li) é usado para representar um item em uma lista. Ele deve estar contido em um elemento pai: uma lista ordenada (ol), uma lista não ordenada (ul) ou um menu (menu). Em menus e listas não ordenadas, os itens de lista geralmente são exibidos usando marcadores. Em listas ordenadas, elas geralmente são exibidas com um contador ascendente à esquerda, como um número ou letra.

Tag "a"

O elemento HTML (a) (ou elemento âncora), com seu atributo href, cria um hiperlink para páginas da Web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outra coisa que uma URL possa abordar. O conteúdo dentro de cada um deve indicar o destino do link. Se o atributo estiver presente, pressionar a tecla enter enquanto estiver focado no elemento irá ativá-lo.(a)href(a)

Tag "hr"

O elemento HTML (hr) representa uma quebra temática entre elementos de nível de parágrafo: por exemplo, uma mudança de cena em uma história ou uma mudança de tópico dentro de uma seção.

Tag "sub"

O elemento HTML (sub) especifica texto embutido que deve ser exibido como subscrito por razões exclusivamente tipográficas. Os subscritos são normalmente renderizados com uma linha de base reduzida usando texto menor.

Tag "sup"

O elemento HTML (sup) especifica texto embutido que deve ser exibido como sobrescrito por razões exclusivamente tipográficas. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando texto menor.

Tag "blockquote"

O elemento HTML (blockquote) indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo (consulte Notas para saber como alterá-lo). Uma URL para a fonte da citação pode ser dada usando o atributo, enquanto uma representação de texto da fonte pode ser dada usando o elemento (cite) cite

Tag "font"

O elemento HTML (font) define o tamanho, a cor e a face da fonte para seu conteúdo. Esse recurso não é mais recomendado. Embora alguns navegadores ainda possam suportá-lo, ele pode já ter sido removido dos padrões da Web relevantes, pode estar em processo de ser descartado ou pode ser mantido apenas para fins de compatibilidade

Tag "del"

O elemento HTML (del) representa um intervalo de texto que foi excluído de um documento. Isso pode ser usado ao renderizar "rastrear alterações" ou informações de comparação de código-fonte, por exemplo. O elemento (ins) pode ser usado com a finalidade oposta: indicar o texto que foi adicionado ao documento.

Tag "abbr"

O elemento HTML (abbr) representa uma abreviação ou acrônimo. Ao incluir uma abreviatura ou sigla, forneça uma expansão completa do termo em texto simples no primeiro uso, juntamente com a marcação da abreviatura. Isso informa ao usuário o que significa a abreviatura ou sigla.(abbr) O atributo title opcional pode fornecer uma expansão para a abreviação ou acrônimo quando uma expansão completa não está presente. Isso fornece uma dica aos agentes do usuário sobre como anunciar/exibir o conteúdo enquanto informa a todos os usuários o que a abreviação significa. Se presente, deve conter esta descrição completa e nada mais.title
